# Structured Data

## Overview

Structured data is generated as JSON-LD and injected into page metadata. This helps search engines understand the content and enables rich snippets in SERP.

## Schemas Implemented

### Article Schema

Used on article detail pages.

```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Article Title",
  "description": "Article excerpt",
  "url": "https://millionblogs.com/article/blog/article",
  "image": "https://example.com/image.jpg",
  "author": {
    "@type": "Person",
    "name": "Author Name"
  },
  "datePublished": "2024-01-01T00:00:00Z",
  "dateModified": "2024-01-02T00:00:00Z",
  "publisher": {
    "@type": "Organization",
    "name": "Blog Name",
    "url": "https://millionblogs.com/blog/blog"
  },
  "mainEntityOfPage": {
    "@type": "WebPage",
    "@id": "https://millionblogs.com/article/blog/article"
  },
  "inLanguage": "en"
}
```

### Blog Schema

Used on blog pages.

```json
{
  "@context": "https://schema.org",
  "@type": "Blog",
  "name": "Blog Name",
  "description": "Blog description",
  "url": "https://millionblogs.com/blog/blog",
  "inLanguage": "en",
  "publisher": {
    "@type": "Organization",
    "name": "MillionBlogs",
    "url": "https://millionblogs.com"
  }
}
```

### Breadcrumb Schema

Used on all pages with navigation context.

```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "https://millionblogs.com"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Blog",
      "item": "https://millionblogs.com/blog/blog"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "Article",
      "item": "https://millionblogs.com/article/blog/article"
    }
  ]
}
```

### Organization Schema

Used on the home page.

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "MillionBlogs",
  "url": "https://millionblogs.com",
  "description": "Discover great content from blogs around the world.",
  "sameAs": []
}
```

## Generator Service

All schemas are generated by `StructuredDataService` (`src/seo/structured-data.service.ts`).

## API Endpoints

```
GET /seo/home            → Organization schema
GET /seo/article/...     → Article + Breadcrumb schemas
GET /seo/blog/...        → Blog + Breadcrumb schemas
GET /seo/category/...    → Breadcrumb schema
```
