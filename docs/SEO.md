# SEO Module

## Overview

The SEO module provides search engine optimization infrastructure for MillionBlogs. It generates sitemaps, robots.txt, canonical URLs, hreflang tags, metadata (Open Graph, Twitter Cards), and structured data (JSON-LD).

## Components

| Service               | Responsibility                                      |
|-----------------------|------------------------------------------------------|
| SitemapService        | XML sitemap generation (main, articles, blogs, etc.) |
| RobotsService         | Dynamic robots.txt generation                        |
| CanonicalService      | Canonical URL generation for all entity types        |
| HreflangService       | hreflang tag generation for multilingual pages       |
| MetadataService       | SEO metadata (title, description, OG, Twitter)       |
| StructuredDataService | JSON-LD schema generation (Article, Blog, etc.)      |

## URL Structure

| Pattern                    | Entity    |
|----------------------------|-----------|
| /article/{slug}            | Article   |
| /blog/{slug}               | Blog      |
| /category/{slug}           | Category  |
| /language/{code}           | Language  |

## Slugs

- **Unique** — Each slug is unique within its entity type
- **Immutable** — Slugs are not changed after creation
- **Slug History** — `SlugHistory` model tracks past slugs for future redirect support

## Canonical URLs

Generated by `CanonicalService`:

```typescript
generateArticleUrl(blogSlug, articleSlug)  // https://millionblogs.com/article/blog/article
generateBlogUrl(blogSlug)                   // https://millionblogs.com/blog/blog
generateCategoryUrl(categorySlug)           // https://millionblogs.com/category/tech
generateLanguageUrl(language)               // https://millionblogs.com/language/en
```

## hreflang

Generated by `HreflangService` for:
- **Articles** — Alternate language versions with `x-default` fallback
- **Blogs** — Alternate language versions with `x-default` fallback

## Metadata

Generated by `MetadataService` for each page type:

| Field          | Source                          |
|----------------|---------------------------------|
| title          | Entity title + blog/app name    |
| description    | Excerpt or fallback text        |
| canonicalUrl   | Full canonical URL              |
| ogTitle        | Same as title                   |
| ogDescription  | Same as description             |
| ogImage        | Featured image URL              |
| ogType         | article / website               |
| twitterCard    | summary_large_image / summary   |

## API

```
GET /seo/home                    - Home page SEO
GET /seo/article/:blogSlug/:slug - Article SEO
GET /seo/blog/:blogSlug          - Blog SEO
GET /seo/category/:slug          - Category SEO
```
